# Teams App Setup

## Required Icons

Please add the following icon files to this folder:

1. **color.png** - 192x192 pixels, full color app icon
2. **outline.png** - 32x32 pixels, white outline only (transparent background)

## Placeholder BOT_ID

Replace `{{BOT_ID}}` in manifest.json with your actual Azure Bot App ID (GUID format).

Example: `12345678-1234-1234-1234-123456789012`
